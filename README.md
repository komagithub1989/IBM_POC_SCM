# IBM_POC_SCM
Basic supply chain management system that tracks inventory levels across multiple warehouses. It should provide functionality to transfer products between warehouses, check stock levels,    and generate reports. 


Explanation How to start application:
from main folder(package.json) just run "npm run start:all"
package.json in IBM_POC_SCM folder
concurrently runs multiple commands at once.
-k kills all when one exits.
-n and -c just colorize output for clarity.


npm install -g concurrently
